# Documentation Index

Este directorio centraliza el contexto operativo para humanos y agentes de IA.

## Archivos
- `docs/AGENT_CONTEXT.md`: contexto práctico del proyecto (cómo arrancar, arquitectura, reglas y convenciones).
- `docs/PLAN_STATUS.md`: avance real contra el `plan-maestro-v1.md` y fases cerradas/pendientes.
- `docs/NEXT_STEPS.md`: backlog accionable para continuar por fases sin perder orden.
- `docs/SECURITY_CHECKLIST.md`: checklist de seguridad para despliegue/release.
- `docs/RUNBOOK.md`: runbook de incidencias operativas.
- `docs/API_QA_COLLECTION.http`: colección de requests para QA/operación.
- `docs/DOD_V1.md`: Definition of Done backend v1.
